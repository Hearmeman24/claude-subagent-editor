# Claude Subagent Editor

Web-based editor for Claude Code subagent YAML configuration files.

## Installation

```bash
uv sync
```

## Usage

```bash
uv run claude-subagent-editor --help
```

## Development

```bash
# Install with dev dependencies
uv sync --all-extras

# Run with auto-reload
uv run claude-subagent-editor --reload
```
